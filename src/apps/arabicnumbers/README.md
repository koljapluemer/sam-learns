# Arabic Numbers Practice

Learn to read the Arabic numbers from 0 to 100.

Imported from [linguanodon](https://github.com/koljapluemer/linguanodon)'s
`arabicnumbers` Django app - see `docs/linguanodon-import.md` at the repo
root for how that import works and how to replicate it for other apps.

Content (`public/data/arabicnumbers/numbers.json`) is a one-time export from
linguanodon's sqlite3 database via `cms/arabicnumbers/import_from_linguanodon.py`.
Rerun that script by hand if the upstream content changes - there's no
ongoing sync.
