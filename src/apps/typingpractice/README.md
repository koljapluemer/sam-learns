# Vietnamese Typing Practice

Practice Vietnamese TELEX/VNI keyboard input.

Imported from [linguanodon](https://github.com/koljapluemer/linguanodon)'s
`typingpractice` Django app - see `docs/linguanodon-import.md` at the repo
root for how that import works and how to replicate it for other apps.

Unlike most imported apps, this one has no database - its word list
(`public/data/typingpractice/vie.json`) was copied verbatim from
linguanodon's `typingpractice/data/vie.json`, no export script needed.
